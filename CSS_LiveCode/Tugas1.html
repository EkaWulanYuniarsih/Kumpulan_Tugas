Praktikum Pemrograman Web 1

Praktikum Pemrograman Web memberikan Ketrampilan kepada mahasiswa dalam pembuatan situs web yang dinamis dan dapat berinteraksi dengan user. Materi meliputi: Teknologi Web, Pembuatan situs, HTML, CSS, PHP, MySQL, dan Visual Studio Code.

Keterangan:

  1. Buatlah class bernama .dasheduntuk heading utama
  2. Guanakan selector div untuk memebuat Box Model
  3. Buatlah class bernama underline-reduntuk kalimat dengan garis bawah merah
  4. Buatlah class bernama shadowuntuk memberikan efek bayangan pada teks
